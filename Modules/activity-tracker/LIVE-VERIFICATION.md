## 🔧 Live Troubleshooting

### **Common Live Issues:**

#### **Issue: Data Not Appearing in Dashboard**
**Symptoms:**
- Device shows online but no activity data
- Screenshots not updating
- Last sync timestamp old

**Solutions:**
1. Check client application status
2. Verify API connectivity
3. Restart tracking service
4. Check network connection

#### **Issue: Intermittent Uploads**
**Symptoms:**
- Data appears sporadically
- Queue count fluctuates
- Connection errors in logs

**Solutions:**
1. Check internet stability
2. Verify server availability
3. Review API rate limits
4. Check firewall settings

#### **Issue: High Resource Usage**
**Symptoms:**
- CPU usage > 20%
- Memory usage increasing
- System performance impact

**Solutions:**
1. Restart application
2. Check for memory leaks
3. Verify screenshot settings
4. Update to latest version
